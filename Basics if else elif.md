## if elif else
в питоне возможно проверять условия. базовый синтаксис:
```Python
if condition_1:
    # do something
elif condition_2:
    # do something else
else:
    # default action
```
## Truthy и Falsy
В Питоне не обязательно передавать в `if` строгое булево значение (`True` или `False`). Под капотом `if` автоматически вызывает встроенную функцию `bool()` для любого переданного объекта.
под капотом bool() ищет функцию `__bool()__` и если ее нет то лезет в `__len()__`
```Python
my_list = []

# ❌ Плохо (не по-питонячьи):
if len(my_list) == 0: ...
if my_list == []: ...

# ✅ Отлично (используем Falsy):
if not my_list:
    print("Список пуст!")
```
