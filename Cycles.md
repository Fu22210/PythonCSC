В Питоне **нет** классического цикла `for (int i = 0; i < n; i++)`. Питоновский `for` — это всегда `foreach`. Он не умеет работать с индексами напрямую, он работает только с объектами, которые поддерживают **протокол итератора**.

**Что делает Питон, когда видит `for x in array:`**

1. Вызывает функцию `iter(array)`. Она возвращает специальный объект-итератор, который помнит, на каком элементе он сейчас находится.
2. Запускает бесконечный невидимый цикл `while True`.
3. На каждом шаге вызывает функцию `next(iterator)`, чтобы получить следующий элемент и положить его в `x`.
4. Как только элементы заканчиваются, итератор бросает скрытое исключение `StopIteration`. Питон ловит его, гасит ошибку и тихо завершает цикл.

цикл while работает проще.  он работает итеративно пока условие выполняется.

*Пример*
```Python
epsilon = 0.001
step_size = 1.0

# Читается как: ПОКА step_size больше epsilon, делай то, что внутри
while step_size > epsilon:
    print(f"Делаем шаг оптимизации... размер шага: {step_size}")
    # Уменьшаем шаг, чтобы цикл когда-нибудь закончился
    step_size = step_size / 2 

print("Алгоритм сошелся!")
```
*Пример с for*
```Python
# У нас есть коробка с гиперпараметрами (learning rates)
learning_rates = [0.1, 0.01, 0.001]

# Читается как: ДЛЯ КАЖДОГО параметра ИЗ списка learning_rates
for lr in learning_rates:
    print(f"Обучаем модель с шагом: {lr}")
```

## break and continue
break - команда которая останавливает тот цикл в котором мы находимся

```Python
targ = 92
found = False
for item in items:
	if item == targ:
		print("ПОБЕДА")
		found = True
		break # тут мы уже все нашли, можно выбрасываться
if not found:
	print("Not ПОБЕДА")
```

else можно писать и с циклами while и for - выполняется, когда мы вываливаемся не по break

```Python
# Например, проверяем сходимость алгоритма по записанной траектории
trajectory = [10.5, 4.2, 1.1, 0.05, 0.001]
target_minimum = 0.0

for point in trajectory:
    if point == target_minimum:
        print("Точный минимум найден, останавливаемся!")
        break
else:
    # Сработает ТОЛЬКО если break ни разу не вызвался (прошли весь массив)
    print("Алгоритм не сошелся к нулю за отведенное число шагов.")
```
continue пропускает одну итерацию цикла

```Python
epochs = [1, 2, 3, 4, 5]

for epoch in epochs:
    if epoch == 2:
        print("Эпоха 2 сломалась, пропускаем!")
        continue  # Перепрыгиваем сразу на эпоху 3
        
    if epoch == 4:
        print("Хватит обучать, переобучились! Выходим.")
        break     # Полностью убиваем цикл
        
    print(f"Успешно завершена эпоха {epoch}")

# Вывод будет: 1, (сообщение о пропуске 2), 3, (сообщение о выходе)
```
